Utilizando Machine Learning para prever os sobreviventes do desastre do Titanic

Desenvolvido para aprimorar minhas técnicas em Análise de dados, Machine Learning e Deep Learning

Esta é a lendária competição do Titanic. O objetivo da competição é dado um conjunto de treino, tentar prever quais foram os sobreviventes do naufrágio. Foi o meu primeiro contato com Machine Learning, desde então cai de amores por essa área da ciência da computação. Desde meu início com ML, já voltei e refiz algumas vezes esse desafio com o objetivo de testar meus conhecimentos a medida que eu evoluo na area.

Atualmente consegui uma solução com 79% de acurácia, e estou no top 700 da competição.

  • Tecnologias: Python para Machine Learning
  • Algoritmos: KNN (Adaboost, Bagging Classifier), Decision Tree, Random Forest, Extra Trees, Gradient Boosting, Naive Bayes, Logistic Regression, SVM, MLP e Tensorflow 2
